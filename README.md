# Samsung Supply Chain & Sales Analytics Dashboard (Power BI)

This project showcases an end-to-end Supply Chain and Sales Analytics solution built in Power BI, using a well-structured star schema data model and interactive dashboards. The model integrates key business areas including Sales, Inventory, Procurement, Production, Shipment, Customers, Suppliers, Products, Facilities, and Date dimensions.

The dashboard provides a high-level overview of total revenue, profit, order quantity, inventory value, shipment performance, and delivery quantity, along with detailed insights such as average lead time by supplier, inventory value by product, shipment delays by carrier, and revenue by customer.

Designed with a modern UI and clear KPIs, this project demonstrates strong skills in data modeling, DAX measures, data visualization, and business insight generation. It is suitable for real-world supply chain, logistics, and enterprise reporting use cases.

Tools & Skills:
Power BI, Data Modeling, DAX, Supply Chain Analytics, Data Visualization, Business Intelligence
